## 📝 Descripción
**Cafe-blog** es un sitio web estático que combina diseño moderno con contenido educativo sobre café.  
Incluye un blog con artículos, cursos gratuitos y talleres, además de un formulario de contacto para interactuar con los visitantes.

---

## 🚀 Características principales
- Página principal con **hero section** y navegación clara.
- Sección de **blog** con artículos sobre beneficios y técnicas del café.
- Sección de **cursos y talleres** con fechas, cupos y precios.
- Formulario de **contacto** con validación básica.
- Diseño responsive y accesible.
- Estilo visual atractivo con imágenes relacionadas al café.

---

## 🛠️ Tecnologías utilizadas
- **HTML5** → estructura semántica.
- **CSS3** → estilos personalizados y responsive.
- **JavaScript (ES6+)** → validación y manejo de eventos.
- **Git & GitHub** → control de versiones y despliegue.
